Flink源码调试（一）

Flink之前调试源码的时候一直有问题，一直无法成功编译，git checkout 到 1.9 版本之后，就可以编译解决了。

附上大佬对maven编译的看法

操作：

首先在bush命令行里面git下载下来，然后切换分支

git checkout release-1.9

这次我们不修改pom文件，将镜像添加到 settings.xml里

直接使用 clean package -DskipTests -Dfast进行编译

mvn命令加上-X可以查看详细文件

1.9的版本会遇到一个问题，JDK版本是有限制的

然后我还遇到了一个npm问题，升级就好了

修改了一个POM，flink-runtime-web的pom里面

改成

install -g -registry=https://registry.npm.taobao.org –cache-max=0 –no-save

贴上maven的setting
